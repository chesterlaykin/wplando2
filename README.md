```
lando start
```

```
lando composer install
```

# Sage theme "wplando2"

## Install composer and npm packages

go to the folder: `web/app/themes/wplando2` and run:

```
lando yarn
```
```
lando yarn build
```
```
lando composer install
```


# Start development server
```
yarn start
```
